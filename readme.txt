Disable notifications for updates to plugins, themes, and version of WordPress


== Description ==

This plugin allows using a simple admin page to disable various notifications of updates to plugins, themes, and new versions of WordPress.

You can disable all notifications or only those you want.

Translation: English, French, Italian, Spanish 

== Installation ==

1. Upload the complete `update-notifications-manager` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Update Notifications Manager' under the 'Settings' tab and configure the plugin


== Frequently Asked Questions ==

= Does it work for WordPress version 2.x.x ? =

No. This plugin was tested and optimized for 3.x.x versions of WordPress.
